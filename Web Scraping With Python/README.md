# Web Scraping Wikipedia and Books to Scrape.com with Beautiful Soup
In this project, I scraped two different sites using Beautiful Soup, turned the results into csv files, and created a visualization of the results. In WikiWebScrapping.ipynb, I collected data on the largest companies by revenue according to Wikipedia and created a graph to visualize which industries were the most profitable. In BookWebScrapping.ipynb, I pulled data from the website books.toscrape.com to collect details on all of the books listed. I then created visuals to show the average price and number of books based on the rating of the book. 

Visualizations were created with matplotlib.
